# EdTech Website

This is a simple, responsive EdTech website for showcasing online courses. It features several pages, including the home page, about us, courses, and contact form. The website is built with HTML, CSS, and JavaScript.

## Features
- **Responsive Design**: The website adjusts seamlessly for different screen sizes, from desktop to mobile.
- **Interactive Forms**: Users can submit messages via a contact form.
- **Course Highlights**: A section showcasing popular courses with descriptions and "Enroll Now" buttons.
